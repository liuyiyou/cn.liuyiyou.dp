Abstract Factory pattern is another creational pattern.

Abstract Factory patterns, also called as Factory of factories, 
have a factory which creates other factories.

When using Abstract Factory pattern, we first use super factory to create factory, 
then use the created factory to create objects.

抽象工厂模式是另一种创建者模式。

抽象的工厂模式，也称为工厂，有一个工厂，创造其他工厂。

当使用抽象工厂模式时，我们首先使用超级工厂来创建工厂，然后使用创建的工厂来创建对象。
